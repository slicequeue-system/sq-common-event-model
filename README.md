## sq-common-event-model

### 📦 개요  
sq-common-event-model은 SliceQueue 시스템 내 다양한 서비스와 모듈에서 공통적으로 사용하는 이벤트 모델(Java 라이브러리)을 제공합니다.  
이 라이브러리는 이벤트 데이터의 표준 구조를 정의하고, 여러 모듈 간 일관된 이벤트 처리와 통신을 지원합니다.

### ✨ 주요 기능 및 특징

- **공통 이벤트 모델 정의**  
  다양한 서비스 및 모듈에서 공통적으로 사용하는 이벤트 객체와 데이터 구조를 정의합니다.  
  이벤트의 타입, 필드, 속성 등이 표준화되어 있어 확장성과 유지보수성이 높습니다.

- **직렬화/역직렬화 지원**  
  이벤트 객체를 효율적으로 직렬화(Serialize) 및 역직렬화(Deserialize)할 수 있는 기능을 제공합니다.  
  JSON 등 다양한 직렬화 포맷 연동이 가능합니다.

- **타입 안전성 보장**  
  이벤트 데이터의 타입을 엄격하게 관리하여, 런타임 오류를 최소화하고 신뢰성 있는 데이터 교환이 가능합니다.

- **확장성**  
  새로운 이벤트 타입 및 속성 추가가 용이하도록 설계되어, SliceQueue 시스템의 성장에 맞춘 유연한 구조를 제공합니다.

### 🛠️ 사용 예시

- 여러 마이크로서비스 및 백엔드 모듈에서 sq-common-event-model을 의존성(Dependency)으로 추가하여,  
  동일한 이벤트 구조를 기반으로 메시지 교환 및 이벤트 기반 로직을 구현합니다.
- SliceQueue 시스템의 각 서비스는 이 공통 모델을 통해 이벤트의 생성, 발행, 구독, 처리 과정을 일관되게 수행할 수 있습니다.
